Chapter 1 Structures and Pointers

Question 1

Differentiate between array and structures?

Answer

Arrays Structures
It is a derived data type It is a user defined data type
A collection of same type of data A collection of different types of data
Elements of an array are referenced using the corresponding subscripts Elements of structure are referenced using dot operator \((.)\)
When an element of an array becomes another array, multidimentional array is formed When an element of an structure becomes another structure, nested structure is formed
Array of structures is possible Structure can obtain arrays as elements

Question 4

Which operator is used for dynamic memory allocation ?

Answer

new

Question 5

What is meant by self referential structure ?

Answer

A self referential structure is a structure in which one of the elements is a pointer to the same structure.

Question 6

  1. How does static memory allocation differ from dynamic memory allocation ?
  2. What is memory leak ?

Answer

  1. Static Dynamic
    Variables get allocated permanently Variables get allocated while program unit is active
    Allocation done before program execution Allocation done during program execution
    Less efficient More efficient
    No memory reusability Memory reusability
    Memory cannot be freed Memory can be freed
    No special operators used Special operators new and delete used for memory allocation and release
    Eg: arrays Eg: Linked list

  2. Memory leak is a situation whick occurs when programmers allocate by using the new operator and forgets to deallocate
