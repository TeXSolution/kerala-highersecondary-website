Chapter 1 Relations and Functions

Question 1

Let \(A=\{x,y,z\}\) and \(B=\{a,b,c,d\}\). Which one of the following is not a relation from \(A\) to \(B\) ?

  1. \(\{(x,a),(x,c)\}\)
  2. \(\{(y,c),(y,d)\}\)
  3. \(\{(z,a),(z,d)\}\)
  4. \(\{(z,b),(y,b),(a,d)\}\)
  5. \(\{(x,c)\}\)

Answer

\(\{(z,b),(y,b),(a,d)\}\)

Question 2

Let \(\mathrm{R}\) be the relation in the set \(\mathbb{N}\) given by \(\mathrm{R}=\{(a,b)\colon a=b-2,b>6\}\). Choose the correct answer.

  1. \((2,4)\in \mathrm{R}\)
  2. \((3,8)\in \mathrm{R}\)
  3. \((6,8)\in \mathrm{R}\)
  4. \((8,7)\in \mathrm{R}\)

Answer

\((6,8)\in \mathrm{R}\)

Question 3

Let \(\mathrm{R}\) be a relation in the set \(\{1,2,3\}\) defined by \(\mathrm{R}=\{(1,1),(1,3),(3,3)\}\). Then the ordered pair to be added to \(\mathrm{R}\) to make it a relation is \(\dots\dots\dots\)

Answer

\((2,2)\)

Question 4

A relation in a set \(\mathrm{A}\) is called \(\dots\dots\dots\) relation, if each element of \(\mathrm{A}\) is related to itself.

Answer

Reflexive

Question 5

What is the minimum number of ordered pairs to form a non-zero reflexive relation on a set of \(n\) elements?

Answer

\(n\) ordered pairs.

Question 6

A relation \(\mathrm{R}\) in a set \(\mathrm{A}\) is called \(\dots\dots\), if \((a_1,a_2)\in \mathrm{R}\) implies \((a_2,a_1)\in \mathrm{R}\) for all \(a_1,a_2\in \mathrm{A}\).

Answer

symmetric

Question 7

Let \(\mathrm{R}\) be a relation on a set \(\mathrm{A}=\{1,2,3\}\), defined by \(\mathrm{R}=\{(1,1),(2,2),(3,3),(1,3)\}\). Then the ordered pair to be added to \(\mathrm{R}\) to make it a smallest equivalence relation is

  1. \((2,1)\)
  2. \((3,1)\)
  3. \((1,2)\)
  4. \((1,3)\)

Answer

\((3,1)\)

Question 8

Which of the following relations on \(\mathrm{A}=\{1,2,3\}\) is an equivalence relation?

  1. \((1,1),(2,2),(3,3)\)
  2. \((1,1),(2,2),(3,3),(1,2)\)
  3. \((1,1),(3,3),(1,3),(3,1)\)
  4. None of these

Answer

\((1,1),(2,2),(3,3)\)

Question 9

Let \(\mathrm{R}\) be a relation defined on \(\mathrm{A}=\{1,2,3\}\) by \(\mathrm{R}=\{(1,3),(3,1),(2,2)\}\). \(\mathrm{R}\) is

  1. Reflexive
  2. Symmetric
  3. Transitive
  4. Reflexive but not transitive

Answer

Symmetric

Question 10

Let \(\mathrm{R}\) be the relation in the set \(\{1,2,3,4\}\) given by \(\mathrm{R}=\{(1,2),(2,2),(1,1),(4,4),(1,3),(3,3),(3,2)\}\). Choose the correct answer.

  1. \(\mathrm{R}\) is reflexive and symmetric but not transitive.
  2. \(\mathrm{R}\) is reflexive and transitive but not symmetric.
  3. \(\mathrm{R}\) is symmetric and transitive but not reflexive.
  4. \(\mathrm{R}\) is an equivalence relation.

Answer

\(\mathrm{R}\) is reflexive and transitive but not symmetric.

Question 11

Consider ther relation \(\mathrm{R}\) in the set \(\mathbb{N}\) of natural numbers difined as \(\mathrm{R}=\{(a,b)\colon ab \text{ is a factor of } 6\}\). Determine whether the relation is reflexive, symmetric or transitive

Answer

\(1,2,3,6\) are factor of 6

$$\begin{align}\mathrm{R}&=\{(1,1),(1,2),(2,1),(1,3),(3,1),\\ &\qquad(6,1),(1,6),(2,3),(3,2)\}\end{align}$$

\(2\in\mathbb{N}\) but \((2,2)\notin\mathrm{R}\qquad\) Not reflexive

\((1,2),(2,1),(1,3),(3,1),(6,1),(1,6),(2,3),(3,2)\qquad\) Symmetric

\((2,3)\in \mathrm{R}\quad\) \((3,2)\in \mathrm{R}\quad\) but \((2,2)\notin \mathrm{R}\qquad\) Not transitive
